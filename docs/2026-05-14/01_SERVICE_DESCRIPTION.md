# 서비스 설명서

작성일: 2026-05-14

## 1. 서비스 개요

SSAFY Home은 공공데이터와 기존 SSAFY Home DB를 활용해 부동산 정보를 제공하는 웹 서비스입니다.  
사용자는 지역, 단지, 거래월, 거래유형을 기준으로 부동산 정보를 조회할 수 있고, 로그인 후 관심지역을 저장할 수 있습니다. 운영자는 관리자 대시보드에서 DB 현황과 작업 로그를 확인할 수 있습니다.

## 2. 서비스 대상

| 사용자 | 사용 목적 |
|---|---|
| 일반 사용자 | 아파트 단지 검색, 실거래 조회, 관심지역 저장, 공지 확인 |
| 학습/개발자 | Spring MVC, JSP, MyBatis, Interceptor, AOP, 외부 API 연동 학습 |
| 운영자 | 데이터 수집 현황, DB 테이블 상태, 작업 로그, 오류 상황 점검 |

## 4. 일반 사용자 기능

### 4.1 실거래 조회

사용자는 거래유형, 법정동 코드, 계약월, 주택명으로 실거래 데이터를 검색합니다.

지원 거래유형:

| 코드 | 의미 |
|---|---|
| `APT_TRADE` | 아파트 매매 |
| `APT_RENT` | 아파트 전월세 |
| `RH_TRADE` | 연립다세대 매매 |
| `RH_RENT` | 연립다세대 전월세 |

기능:

- 선택 거래유형 데이터 수집
- 4개 거래유형 통합 수집
- DB에 저장된 실거래 검색
- 거래유형별 월간 요약
- 평균 거래가, 평균 보증금, 평균 월세, 평균 면적, 면적당 가격 표시

### 4.2 아파트 단지 검색

기존 DB의 `dongcodes`, `houseinfos`, `housedeals`를 기반으로 단지 정보를 검색합니다.

기능:

- 지역명 또는 법정동 코드로 동 검색
- 단지명, 도로명, 지역명으로 단지 검색
- 단지별 거래 건수와 최근 거래 정보 표시
- 단지 상세 정보와 최근 거래 이력 표시

### 4.3 회원 및 세션

로그인은 `HttpSession` 기반입니다.  
로그인 성공 시 `LOGIN_MEMBER_ID`가 세션에 저장됩니다.

기능:

- 회원가입
- 로그인
- 로그아웃
- 내 정보 조회
- 회원 목록 조회

### 4.4 관심지역

로그인한 사용자는 관심지역을 저장하고 나중에 다시 조회할 수 있습니다.

특징:

- 로그인 필수
- 회원별 데이터 분리
- 삭제 시 `id + member_id` 조건으로 소유권 보장

### 4.5 공지사항

공지사항은 서비스 안내, 변경사항, 데이터 수집 관련 메시지를 관리하기 위한 기능입니다.

기능:

- 공지 목록 검색
- 공지 상세 조회
- 공지 작성
- 공지 수정
- 공지 삭제
- 조회수 증가

## 5. 관리자 기능

관리자 메뉴는 일반 상단 네비게이션에서 숨겨져 있습니다.  
페이지 최하단 footer 오른쪽의 작은 `관리자` 링크로 접근합니다.

관리자 페이지는 로그인 필요 기능입니다.

제공 기능:

| 기능 | 설명 |
|---|---|
| 요약 metric | 회원 수, 수집 실거래 수, 단지 수, 관심지역/공지 수 |
| 거래 유형별 적재 현황 | `property_deals`의 deal_type별 건수 |
| 최근 작업 로그 | 회원/관심지역/공지/수집 작업 로그 |
| 현재 DB 조회 | 허용된 테이블을 선택하고 키워드로 필터링 |
| 최근 수집 실거래 | 최신 `property_deals` 8건 표시 |

관리자 DB 조회 허용 테이블:

- `members`
- `property_deals`
- `houseinfos`
- `housedeals`
- `dongcodes`
- `favorites`
- `notices`

## 6. 외부 API 사용

| 서비스 | 사용 목적 | 담당 코드 |
|---|---|---|
| data.go.kr | 실거래 데이터 수집 | `DealService` |
| VWorld | 행정구역 API 응답 확인 | `RegionService` |
| SGIS | 토큰 발급 확인 | `RegionService` |

외부 API 실패는 `ExternalApiException`으로 처리되며, 브라우저에는 502 JSON 오류가 반환됩니다.

- JSP ViewResolver
- MyBatis SQL 관리
- 외부 공공 API 호출
- XML 파싱
- 세션 로그인
- Interceptor 권한 처리

- 파일 로그 기반 운영 추적
